# 🎬 BSI CiniPass – Movie Ticket Booking System

## 📌 Introduction
**BSI CiniPass** is a simple **Python-based movie ticket booking system** developed as a **micro project**.  
The project uses **Tkinter** to create a graphical user interface (GUI) and demonstrates core Python programming concepts such as lists, dictionaries, tuples, loops, and decision making.

---

## 🛠️ Technologies Used
- Python 3
- Tkinter (GUI library)
- datetime module

---

## 🎯 Project Features
- Graphical User Interface (GUI)
- List of available movies with genre and rating
- Multiple showtime options
- Seat categories with different prices
- Sidebar navigation menu
- Booking history display
- Simple and user-friendly design

---

## 📚 Python Concepts Used
- Lists
- Dictionaries
- Nested Dictionaries
- Tuples
- Functions
- Loops
- Conditional Statements
- GUI Development using Tkinter
- Event Handling
